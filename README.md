# E-Library Management System

A full-stack web application for managing books in an online library.

## Features

- User Registration
- User Login
- JWT Authentication
- Add Books
- View Books
- Search Books
- Update Books
- Delete Books
- Logout Functionality

## Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JWT
- bcrypt

### Media Storage
- Cloudinary
